
# CoastCrave 
A modern, responsive online seafood ordering website with a complete management system for customers, staff, and administrators.

## Overview
CoastCrave brings the seafood dining experience online with a clean, ocean-inspired design and smooth, intuitive interactions. Built with HTML, Tailwind CSS, and JavaScript, it offers a seamless ordering flow for customers while providing powerful, role-based dashboards for staff and admins to manage daily operations efficiently.

## Features

### Customer Experience
* Clean, ocean-themed interface with smooth animations
* Interactive menu with category filters
* Shopping cart & seamless checkout
* Real-time order tracking
* Customer reviews & rating system

### Staff Dashboard
* Manage incoming orders in real time
* Update order status (Pending → Delivered)
* Search, filter, and organize orders
* Fully responsive for mobile and desktops

### Admin Dashboard
* Can add, edit, and remove menu items across categories
* Can add, edit, and remove users across categories
* Manage users and assign roles
* View analytics with interactive charts
* Monitor sales performance and trends

## Tech Stack

* Frontend: HTML, Tailwind CSS, JavaScript
* Storage: localStorage for saving data
* Extras: Smooth UI animations, shimmer effects, and modern components

## User Roles
* Customer – Can browse the menu, place orders, track deliveries
* Staff – Handle and update customer orders
* Admin – Manage the entire system, users, menu, and analytics

## Getting Started
1. Download or clone the project
2. Open `index.html`in any browser
3. Explore the customer, staff, and admin sections
